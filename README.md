# 生命倒计时

一个富有意义的网页应用，用于记录和可视化个人生命历程。

## 功能特点

- **生命网格**：以月为单位展示从出生到100岁的完整生命历程
- **实时更新**：当前月份实时显示已过去的时间比例
- **统计数据**：清晰展示已度过月份、剩余月份和当前年龄
- **视觉反馈**：
  - 已度过的月份呈灰色
  - 当前月份以渐变色高亮显示，带有脉冲动画
  - 未来月份保持明亮
  - 每个格子内部动画展示当月天数的消逝

## 使用方法

1. 直接在浏览器中打开 `index.html`
2. 如需修改出生日期，编辑 `script.js` 中的 `BIRTH_DATE` 变量

## 设计理念

通过可视化生命的流逝，提醒我们：
- 时光珍贵，不可逆转
- 珍视当下每一刻
- 合理规划人生目标
- 活出有意义的生命

## 技术实现

- 纯前端实现，无需服务器
- 响应式设计，支持各种屏幕尺寸
- 平滑动画和交互效果
- 实时计算和更新
